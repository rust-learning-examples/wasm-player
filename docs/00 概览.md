### 基础命令
```bash
# 创建项目
wasm-pack new wasm-player
wasm-pack build
# 创建前端页面引用wasm
npm init wasm-app www

# 编译项目 并进入到pkg, link包 https://llever.com/rustwasm-book/game-of-life/hello-world.zh.html
npm link 
# 进入www
npm link wasm-player
# 修改www/index.js
# 启动， 并访问： http://localhost:8080/
npm start
```
